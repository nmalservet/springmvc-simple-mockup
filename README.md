# springmvc-simple-mockup
A simple mockup to start a project with springmvc. 

##Dependencies used are:
* spring-boot-starter-web 1.4.0

##Requirements

* Java 8 installed
* Maven
* A good source code editor ( I used eclipse or netbeans. You can find all tools needed in Netbeans 8.1 with the package "ALL")

##How to install
Clone this repository on your computer
Run maven clean install
Execute the following maven command
mvn spring-boot:run
Verify in your broser if it's works:
http://localhost:8080
and the other path
http://localhost:8080/questionnaire
will show you : Yes you are on the right path
